# Polymarket Codex

Projeto inicial em TypeScript + Node.js (Express) para construção da ferramenta de trading Polymarket.

## Scripts

- `npm run dev`: sobe servidor em modo watch
- `npm run build`: compila TypeScript para `dist/`
- `npm run start`: executa build compilado
- `npm run lint`: executa lint
- `npm run test`: executa testes unitários iniciais
